Sub ExportSlides()
  For i = 1 To ActiveWindow.Selection.SlideRange.Count
    Dim fileName As String
    If (i > 10) Then
      fileName = "Slide" & CStr(i) & ".png"
    Else
      fileName = "Slide0" & CStr(i) & ".png"
    End If
    ActiveWindow.Selection.SlideRange(i).Export fileName, "PNG", 2400, 1800
  Next
End Sub

Service Spec JSON Tool
======================

template
  - lists order of worship
  - leader position ("Song Leader")
  - Song References ("song-1", "song-2")

schedule
  - contains list of services on a given date (sunday, wednesday)
  - time, language, template for each service
  - Includes service type ("Sun AM") for Congregate lookup

style
  - contains background image names for different elements of worship
  - other future adjustments?


[service-id]-songs
  - for each reference ("song-1"), provides song details

[service-id]-leaders
  - fetched from api.embryhills.church OR manual
  - for each leader position, lists leader name
  - include preacher

[service-id]-readings
  - list scripture readings, LS readings, Collection readings?
  - sermon topics

[service-id]-spec
  - date, time
  - language
  - select service-template
  - service type ("Sun AM") for Congregate lookup

[service-id]
  - final output of JSON compiler
  - input to slides20.py to generate slide deck
